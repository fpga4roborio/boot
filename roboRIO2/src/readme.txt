uboot.txt contains the minimal uboot environment variables to boot PetaLinux (with root extracted to the second partition, formatted in ext4).
uboot.org.txt contains the original uboot environment variables from the official SD image. For reference only.
ni-7AAE.dts is the decompiled device tree source from ni-7AAE.dtb (for roboRIO 2.0). For serious editing, original source at https://github.com/ni/linux/blob/nilrt-academic/master/4.14/arch/arm/boot/dts/ni-7AAE.dts should be used instead.
